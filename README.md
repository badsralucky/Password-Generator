A sleek and functional web-based password generation tool built with a focus on ease of use and security. This application allows users to create strong, customizable passwords with various complexity settings and a real-time strength indicator.

🚀 Features
Dynamic Password Length: Generate passwords ranging from 1 to 30 characters using an interactive slider.

Customizable Complexity: Toggle options for:

Lowercase (a-z)

Uppercase (A-Z)

Numbers (0-9)

Symbols (!-$^+)

Include Spaces

Exclude Duplicate characters

Strength Indicator: Visual feedback bar that updates in real-time to categorize your password as Weak, Medium, or Strong.

One-Click Copy: Built-in clipboard integration to quickly copy your generated password.

Responsive Design: A modern UI styled with Poppins typography and Material Symbols.

🛠️ Tech Stack
HTML5: Semantic structure.

CSS3: Custom styling, Flexbox layout, and strength-based transitions.

JavaScript (ES6): Logic for character randomization, strength calculation, and clipboard interaction.

Google Fonts: 'Poppins' and 'Material Symbols Rounded'.

⚙️ Project Configuration
The project includes a .hintrc configuration for development, specifically tailored to handle accessibility linting for form labels. 

📂 File Structure
index.html: The main structure of the application.

style.css: Modern visual styling and animations.

script.js: Interactive functionality and password logic.


.hintrc: Development linter configuration. 

💻 How to Run
Clone this repository to your local machine.

Open the index.html file in any modern web browser.

Adjust the length and settings, then click Generate Password.
